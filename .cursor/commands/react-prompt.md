# react-prompt

## Project Overview

Create a landing page for **Catlife**, a company that provides cat insurance services.

## Figma Design Reference

**Figma Link:** https://www.figma.com/design/7YaKpGYjOiYJor8aM1F8zd/Catlife-Site-1.0--Copy-?node-id=647-825&t=FmikKPTIdMv3JC9V-4

## Implementation Plan

### 1. Import Design from Figma
- Extract design assets, colors, typography, and component specifications from the Figma file

### 2. Create Design System
- **Colors**: Configure using Tailwind CSS 4
- **Components**: Set up using shadcn/ui

### 3. Generate Application Pages
- **Home Page**: Main landing page for Catlife
- **Checkout Page**: Checkout flow for insurance purchases

### 4. Quality Assurance
- **Responsivity**: Analyze and ensure responsive design across all breakpoints
- **Code Quality**: Review code for best practices, maintainability, and performance

## Notes

- Use all available MCP tools for design extraction and implementation
- If any images, icons, or colors cannot be extracted from Figma, notify immediately
- You don't have to manually create the files for the registry, shadcn will do it for you.
